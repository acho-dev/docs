## Install the SDK

The Aden SDK has zero required dependencies. Install optional provider support as needed.

### Basic Installation

```bash
pip install aden
```

### With Provider Support

<CodeGroup>

```bash OpenAI
pip install aden[openai]
```

```bash Anthropic
pip install aden[anthropic]
```

```bash Google Gemini
pip install aden[gemini]
```

```bash All Providers
pip install aden[all]
```

</CodeGroup>

### Framework Integrations

<CodeGroup>

```bash PydanticAI
pip install aden[pydantic-ai]
```

```bash LiveKit Voice Agents
pip install aden[livekit]
```

</CodeGroup>

### Database Backends

<CodeGroup>

```bash TimescaleDB (Async)
pip install aden[timescale]
```

```bash TimescaleDB (Sync)
pip install aden[timescale-sync]
```

</CodeGroup>

### Development

```bash
pip install aden[dev]
```

## Requirements

| Requirement | Version |
|-------------|---------|
| Python | >= 3.10 |

### Optional Dependencies

| Package | Version | Required For |
|---------|---------|--------------|
| `openai` | >= 1.0.0 | OpenAI provider |
| `tiktoken` | >= 0.5.0 | OpenAI token counting |
| `anthropic` | >= 0.20.0 | Anthropic provider |
| `google-generativeai` | >= 0.4.0 | Gemini provider |

## Verify Installation

Create a simple test file:

```python
from aden import instrument, MeterOptions, create_console_emitter, uninstrument, get_instrumented_sdks

# Instrument the SDK
instrument(MeterOptions(
    emit_metric=create_console_emitter(pretty=True),
))

result = get_instrumented_sdks()
print(f"Instrumented: {result}")
# Output: InstrumentationResult(openai=True, anthropic=False, gemini=False)

# Clean up
uninstrument()
```

Run the test:

```bash
python test.py
```

## Virtual Environment (Recommended)

<CodeGroup>

```bash venv
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
.venv\Scripts\activate     # Windows
pip install aden[all]
```

```bash Poetry
poetry add aden
poetry add openai anthropic google-generativeai  # Optional
```

```bash uv
uv pip install aden[all]
```

</CodeGroup>

## Type Checking

The SDK includes type annotations. For type checking with mypy:

```toml
# pyproject.toml
[tool.mypy]
python_version = "3.10"
strict = true
```

```bash
mypy your_code.py
```
