## Overview

Report tools generate financial statements and analysis from your General Ledger data.

## Available Tools

### finance_report_trial_balance

Generate a trial balance report.

**Parameters:**
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `posting_period_id` | string | Yes | Accounting period ID |
| `include_zero_balances` | boolean | No | Include accounts with zero balance |

**Example:**
```json
{
  "tool": "finance_report_trial_balance",
  "parameters": {
    "posting_period_id": "period-2024-01",
    "include_zero_balances": false
  }
}
```
